Nova Royal ## Executive 3D AI Assistant
Nova Royal is a voice-first 3D holographic AI assistant built for executives. She combines a premium presence with real productivity tools to manage email, meetings, and scheduling.
### Key Features
**1. Executive Voice Interface**- 3D Holographic Avatar with royal blue/gold glow- Hold-to-Talk voice commands using Web Speech API- Premium Female Text-to-Speech responses
**2. Smart Email**- AI email draft generation- Reads unread emails aloud- Command: "Check email"
**3. Meeting AI**- Summarize meeting transcripts automatically- Set meetings with voice commands- Command: "Set meeting with [Name] at [Time]"
**4. Auto Scheduler**- View daily schedule at a glance- Command: "Check schedule"
### Design- **Theme**: Royal Executive- **Colors**: Black `#0A0A0A`, Royal Purple `#4F46E5`, Gold `#FBBF24`, Emerald `#10B981`- **UI**: Left sidebar + Dashboard + Hologram widget bottom left that only shows when talking
### Tech Stack- **Frontend**: React + TypeScript + Tailwind CSS- **Built With**: Lovable- **Voice**: Web Speech API - SpeechRecognition + SpeechSynthesis- **Icons**: lucide-react- **Avatar**: MP4 Hologram Video with CSS animations
